# node.js 第十次测验题和答案

## 一、测试：保证代码质量

1. 单元测试的两大形态是什么？
```
TDD:Test-Driven Development 测试驱动开发
BDD:Behavior Driven Development 行为驱动开发
```
2. 用 Node.js 程序写测试脚本，用到了 Node.js 中的哪个模块？
```
assert
```
3. assert 模块中断言函数分为哪三类？
```
肯定性断言
否定性断言
异常类断言
```
4. 断言函数一般有三个参数，分别是什么？
```
1. actual：被测程序实际运行结果
2. expected：被测程序执行的预期结果
3. message：比对信息
```
